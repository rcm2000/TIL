## gitignore

> git으로 관리하지 않을 파일/폴더 등을 관리해줌

- `.gitignore` 파일을 만들어서 관리

  ```
  user.csv # 특정 파일
  venv/ # 특정 폴더
  *.csv # 특정 확장자
  ```

  - 개발 프로젝트시 어떤 것들을 gitignore로 관리를 하는가?
    - OS(운영체제)에서 활용되는 파일들
    - IDE(통합개발환경 - pycharm), Text editor(vs code) 등에서 활용되는 파일
      - 예) pycharm -> .idea/
    - 개발 언어(python) 혹은 프레임워크(django)에서 사용되는 파일
      - 가상환경 - `venv/`
      - `__pycache__/` ...
